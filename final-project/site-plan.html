Individual Project - Site Plan

This planning document outlines the subject, purpode, scenarios, solor schema, typography, and wireframes.

Site Name

Name: Math Courses

Reason:It communicates the site's focus: curated mathematics courses with a placement quiz to guide learners.

Site Purpose

The site provides a hub for learning math

  • Home:Introduces the site and invites to take the diagnostic.
  • Courses: Diagnostic test (Open Trivia DB), personalized recommendation, and courselist.
  • Sign Up Registration form with results page.

Senarios

The site provides a hub for learning math

  • “I struggle with algebra. Where should I start?” → Diagnostic recommends beginner courses.
  • “Can I see only intermediate lessons?” Use level filter buttons above the course grid.

Color Schema

Accent (#20C997) — links, highlights
Text (#0B132B) — body
Background (#F7F9FC) — page background

Typography

  • Body: Inter, regular
